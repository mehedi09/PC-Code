Web Server
You may choose to test the generated application right after the completion of code generation. This feature requires IIS Express installed on your computer.
The web.config file of the generated project is always overwritten. You many need to make changes to the web.config file to link custom component libraries and enter custom entries required for your project. Use AppendChild, InsertBefore, and InsertAfter instructions to make precise custom changes to the generated configuration file of the project. Place instruction followed by ":" and XPath expression leading to existing node followed by XML snippet.
